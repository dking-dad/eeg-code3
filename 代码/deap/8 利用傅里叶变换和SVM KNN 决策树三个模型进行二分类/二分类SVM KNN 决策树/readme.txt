该存储库包含有关从脑电信号中识别情绪的信息，使用的数据集是DEAP数据集，并且用于预处理信号的fft转换。
使用KNN，SVM和决策树等机器学习算法来构建模型，并研究了其各自的准确性。

This repository contains information on emotion recognition from eeg signals 
The data set used is DEAP data set and fft transformation was used for pre-processing the signals. 
Machine learning algorithms such as KNN, SVM and decision tree were used to build models 
and its respective accuracies were investigated.
